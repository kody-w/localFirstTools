## Use This Flowchart For:

1. **Planning Implementation**: Follow the flow to build features in order
2. **Testing Story Logic**: Ensure all branches work correctly
3. **Writing Dialogue**: Know where each character stands emotionally
4. **Debugging Flags**: Track which flags should be set when
5. **Balancing Choices**: Ensure meaningful consequences
6. **Content Gating**: Lock/unlock areas at right times
7. **Player Communication**: Show progress through campaign
