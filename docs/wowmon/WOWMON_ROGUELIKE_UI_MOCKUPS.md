## Color Scheme & Visual Style

### Recommended Colors
- **Background:** Dark blues/purples (#0a0e27, #1a1f3a)
- **UI Panels:** Dark gray with blue borders (#2d3748, #4facfe)
- **Text:** White/cyan (#fff, #00f2fe)
- **Positive:** Green (#4ade80)
- **Warning:** Yellow/Orange (#fbbf24)
- **Danger:** Red (#ef4444)
- **Legendary:** Gold (#ffd700)
- **Epic:** Purple (#a855f7)
- **Rare:** Blue (#3b82f6)
- **Common:** Gray (#9ca3af)

### Animation Touches
- Room selection: Glow effect on hover
- Relic icons: Gentle pulse/shimmer
- Boss encounters: Screen shake
- Death screen: Slow fade to black
- Victory: Confetti particles
- Soul shard gain: Number count-up animation
- Unlocks: Flash/sparkle effect
