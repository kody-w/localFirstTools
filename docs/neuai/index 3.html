Quick Install

Up and running in 60 seconds

1

Copy files to ~/.neuai

mkdir -p ~/.neuai && cp neuai-cli.py neuai-agent-bridge.py ~/.neuai/
2

Run first-time setup

python3 ~/.neuai/neuai-cli.py

You'll be prompted for your Azure OpenAI endpoint, API key, and deployment name.

3

Start using NeuAI

python3 ~/.neuai/neuai-agent-bridge.py chat "Hello, NeuAI!"
4

(Optional) Set up project-local memory

mkdir -p .neuai/data && echo '{"user_guid":"my-project"}' > .neuai/identity.json
